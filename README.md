# Repaso para Segundo Parcial

Este repositorio contiene los archivos de las dos series del repaso para el segundo parcial.

La primera carpeta contiene la primera serie, esta es una calculadora que contiene diferentes botones para realizar diferentes calculos.

La segunda carpeta caution la segunda serie, esta es una aplicacion que hace consulta, modificaciones, anular, eliminar y mostrar datos de una base de datos de forma visual.
